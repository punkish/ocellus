About

Ocellus is a frontend to the Biodiversity Literature Repository (BLR) with images on Zenodo, taxonomic treatments in TreatmentBank, and citations on RefBank. Ocellus depends on Zenodeo, a nodejs API that queries, analyzes and aggregates results from these various repositories via a single, unified interface. Start typing in the search field and Ocellus will suggest from > 5100 taxonomic familes, and then retrieve the results for you.
